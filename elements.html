Projects Accomplished

Data Transfer Automation from local system to distributed storage system

Problem faced: Availability of data during downtime in a particular zone.
Solution: Assembled an ETL pipeline that transfers data from a set of CSV files to Apache Cassandra or dynamodb tables to check continuous data availability using Amazon Web Service. Provides an automatic backup when regional server has a downtime.

Analysis of Spotify data to find the daily trends

Problem faced: To find music trends of a particular user.
Solution: Programmed an ETL pipeline using Spotify API to find trending songs of the day.

Job Advertisement Finder

Problem faced: Daily collation of jobs at a glance.
Solution: Introduced a web crawler using python to gather available jobs according to user from the timesjob.com website, and store them in a text file with only relevant information automatically using a scheduled refresh.

Amazon Price Tracker

Problem faced: Track price drop of an e-commmerce product.
Solution: Saved 100% time during a sale to check if the price of the product has depreciated. Created an email alert when the price of the product has changed by using scrappy library in python.

Storage architeture optimization of ASTROSAT satellite data quality reports

Problem faced: High latency time, low data portability.
Solution: Developed a django-based system with a model-view-template architecture with MySQL database to manage data when the amount of data reached a critical point with an efficiency of 97.2%. Designed an android platform with caching of frequently used pages with the help of Google Volley library for high portability of data.

Use of AWS lambda to find new aspects of serverless computing

Solution: Designed a serverless architecture for a chatr application using trigger based AWS lambda function, thereby decreasing the server maintainance and configuration more than 50%.
